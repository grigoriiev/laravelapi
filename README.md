
## ОПИСАНИЕ ФУНКЦИОНАЛА ДЛЯ ПРОВЕРЯЮЩЕГО

NOTEBOOK
REST API для записной книжки



PUT
/api/v1/notebook/{id}
Обновление записей без картинки



GET
/api/v1/notebook/{id}
Получение записи



DELETE
/api/v1/notebook/{id}
Удаление записи



POST
/api/v1/notebook/{id}
Обновление картинки



POST
/api/v1/notebook
Добавление записи



GET
/api/v1/notebook
Получение всех записей



GET
/api/v1/notebooklist/{limit}
Получение определенного количества записей (вывод информации в списке по странично)
